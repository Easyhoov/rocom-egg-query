## 二、技术架构

### 技术栈

| 组件 | 技术 |
|------|------|
| 后端框架 | FastAPI |
| 服务器 | Uvicorn |
| 数据源 | RocomUID (breeding.json, 1623条) |
| 图片资源 | rocomegg (creature-atlas, 465张webp) |
| 前端 | 原生 HTML/CSS/JS |
| 部署 | your-server-ip:2026 |

### 为什么选原生 HTML（不是 Vue 3）

1. **轻量简洁** — 单文件即可运行，无需构建工具
2. **快速上线** — 开发周期短，适合快速验证
3. **维护简单** — 无依赖，不怕框架升级
4. **适合当前需求** — 功能单一，不需要复杂组件复用
5. **后期可迁移** — 需要时再迁移到 Vue 3

## 三、图片匹配机制

### 匹配链路

```
精灵名称 → rocomegg精灵库（name → id） → 图片文件（{id}-base.webp）
```

### 匹配策略

1. **精确匹配**：直接通过精灵名称匹配
2. **模糊匹配**：
   - 去掉血脉后缀：如"迪莫（光系血脉）" → "迪莫"
   - 去掉异色前缀：如"异色恶魔狼" → "恶魔狼"
   - 去掉首领前缀：如"首领蹦蹦种子" → "蹦蹦种子"

### 匹配结果

| 指标 | 数值 |
|------|------|
| 总记录数 | 1623 条 |
| 匹配成功 | 1512 条（93.2%） |
| 未匹配 | 111 条（rocomegg 数据库中没有） |

## 四、API 接口

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 前端页面 |
| `/api/query` | GET/POST | 孵蛋查询 |
| `/api/pets` | GET | 精灵列表（分页） |
| `/api/stats` | GET | 数据统计 |
| `/creature-atlas/{id}.webp` | GET | 精灵图片 |

### 查询参数

```
GET /api/query?height=0.25&weight=1.5&precious=0
```

- `height` (float, 必填): 身高（米）
- `weight` (float, 必填): 体重（千克）
- `precious` (int, 可选): 蛋类型筛选
  - `0` = 神奇的蛋
  - `1` = 炫彩蛋
  - 不传 = 不限类型

```

│  🥚 孵蛋查询                            │
│  洛克王国：世界                         │
├─────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐             │
│  │ 身高 (m) │  │ 体重 (kg)│             │
│  │  0.24    │  │  1.60    │             │
│  └──────────┘  └──────────┘             │
│                                         │
│  [🥚 神奇的蛋] [✨ 炫彩蛋] [📦 不限类型] │

│            [🔍 查询]                    │
├─────────────────────────────────────────┤
│  [全部 14] [🥚 神奇 14] [✨ 炫彩 0]     │
│  你的R值: 6.00                          │
├─────────────────────────────────────────┤
│  ┌────────┐ ┌────────┐ ┌────────┐      │
│  │ [图片] │ │ [图片] │ │ [图片] │      │
│  │ 小黑猫 │ │ 喵喵   │ │ 伊贝儿 │      │
│  │ R:5.77 │ │ R:5.75 │ │ R:6.30 │      │
│  │ 差:0.23│ │ 差:0.25│ │ 差:0.30│      │
```

### 设计特点

1. **卡片式布局** — 每个精灵一张卡片，3列网格
2. **蛋类型标签** — 彩色角标显示蛋类型
3. **精灵图片** — 实际精灵图片，加载失败降级为 emoji
4. **R值展示** — 清晰显示 R 值和差值
5. **标签页筛选** — 全部/神奇/炫彩 三个标签页
6. **移动端适配** — 响应式设计，小屏幕自动调整
