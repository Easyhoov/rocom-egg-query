✅ 新增页面 Checklist

  • 确定风格归属（紫蓝 or 暖色）
  • 在 router/index.js 添加路由
  • 在 App.vue 导航栏添加入口
  • 在 Home.vue features 添加卡片
  • 创建 views/XxxPage.vue
  • main.py 添加页面路由（返回 INDEX_HTML）
  • 新增静态目录必须在 main.py mount
  • 图片引用用 :src="'/img/xxx.png'" 动态绑定
  • cd frontend && npx vite build
  • 重启 uvicorn 服务

⚡ 设计原则

  • 移动端优先 — 420px 居中，底部留 80px
  • 浅色渐变背景 — 绝不用深色主题
  • 白色圆角卡片 — 16~18px 圆角
  • 紫色主色调 — #8b3dff / #667eea
  • 本地素材优先 — 不依赖外部 CDN
  • 动态绑定图片 — 避免 Vite 构建报错
  • 平滑过渡 — transition: .2s
  • 按压缩放 — transform: scale(0.97)
