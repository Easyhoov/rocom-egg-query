# 子页面 Figma 风格改版方案

## 涉及页面

| 页面 | 路由 | 当前状态 | 核心功能 |
|------|------|----------|----------|
| 孵蛋查询 | /egg-query | Vue SPA | 输入身高体重 → 查匹配精灵 |
| 精灵图鉴 | /compendium | Vue SPA | 浏览、搜索、筛选精灵列表 |
| 蛋组配对 | /egg-group | Vue SPA | 查看蛋组兼容性 |
| 家园炼金 | /garden | Vue SPA | 炼金配方查询 |
| 远行商人 | /merchant | Vue SPA | 商人商品查看 |

## 设计规范（继承首页）

- 字体：Inter + JetBrains Mono
- 颜色：纯黑/纯白 + Figma 四色渐变装饰
- 组件：药丸按钮、圆形图标、渐变条、卡片
- 响应式：560px / 960px 断点
- Nav：所有子页面统一导航栏（Logo + 5个功能链接 + CTA）

## 子页面通用 Nav

```
┌─────────────────────────────────────────────────────────────┐
│ ⚔ 洛克王国助手    孵蛋查询  精灵图鉴  蛋组配对  家园炼金  远行商人  [开始查询] │
└─────────────────────────────────────────────────────────────┘
```

- 首页：Logo + CTA（无链接，卡片导航）
- 子页面：Logo + 5链接 + CTA（全显示，当前页高亮）
